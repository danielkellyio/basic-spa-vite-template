# Basic SPA Vue Vite Template
### Features

* Vue 3
* Vue Router
* Tailwind CSS
* `@` import alias
* ESLint + Prettier (with VS Code Settings)
* Boilerplate Auth User Composable
* Boilerplate Auth Pages

### How To Use

```
$ git clone git@github.com:danielkellyio/basic-spa-vue-vite-template.git
$ npm install
```
